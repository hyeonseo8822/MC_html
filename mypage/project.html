Nugget

  • Project Overview

    • 너겟은 사용자가 하루에 한 번 지식을 얻을 수 있는 앱입니다.
    이 앱은 매일 새로운 정보나 지식을 제공하며, 사용자는 매일 아침마다 간단한 퀴즈나 팁을 통해 지식을 확장할 수 있습니다.

  • Main Features

    • 하루 한 번의 지식 제공: 사용자가 하루에 한 번 새로운 정보를 받을 수 있도록 알림 기능이 구현되어 있습니다.
    퀴즈 및 팁 제공: 사용자는 다양한 분야에 대한 퀴즈나 유용한 팁을 통해 새로운 지식을 얻습니다.

  • Tech Stack

    • 이 프로젝트는 JavaScript, HTML, CSS를 사용하여 웹 기반으로 개발되었습니다. 서버 측에는 간단한 파일 저장 방식을 사용하여 데이터를 관리합니다.

Mini Games

  • Project Overview

    • 미니게임 앱은 슈팅 게임과 지렁이 게임을 포함한 앱입니다.
    이 앱은 간단한 게임들을 제공하여 사용자들이 즐거운 시간을 보낼 수 있도록 합니다.

  • Main Features

    • 슈팅 게임: 사용자는 적을 처치하며 점수를 얻고, 게임에서 승리하기 위한 목표를 가지고 플레이합니다.
    지렁이 게임: 지렁이가 성장하며 장애물을 피하고 음식을 먹어가며 점수를 높입니다. 간단한 컨트롤과 도전적인 요소가 특징입니다.

  • Tech Stack

    • 이 프로젝트는 Java Swing을 사용하여 GUI 기반의 게임으로 개발되었습니다. 각 게임은 키보드 입력을 받아서 캐릭터를 움직이는 방식으로 구현되었습니다.

Notepad

  • Project Overview

    • 메모장 앱은 Java Swing을 사용하여 만든 간단한 메모 앱입니다.
    사용자는 텍스트를 작성하고, 이를 로컬에 저장하여 언제든지 불러올 수 있습니다. 직관적인 인터페이스로 손쉽게 메모를 작성할 수 있습니다.

  • Main Features

    • 메모 작성: 사용자는 텍스트 박스에 내용을 작성하고 저장할 수 있습니다.
    메모 불러오기: 저장된 메모를 앱에서 불러와서 다시 확인하거나 수정할 수 있습니다.

  • Tech Stack

    • 메모장 앱은 Java Swing을 사용하여 GUI를 구성하고, 로컬 저장소에 데이터를 저장하기 위해 파일 시스템을 활용했습니다.
